swoole_server->stats

得到当前Server的活动TCP连接数，启动时间，accpet/close的总次数等信息。

array swoole_server->stats();

返回的结果数组示例：

array (
  'start_time' => 1409831644,
  'connection_num' => 1,
  'accept_count' => 1,
  'close_count' => 0,
);
  • start_time 服务器启动的时间
  • connection_num 当前连接的数量
  • accept_count 接受了多少个连接
  • close_count 关闭的连接数量
  • tasking_num 当前正在排队的任务数

stats()方法在1.7.5+后可用
